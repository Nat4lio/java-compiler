# Java Compiler

A compiler for a Java-like language developed in C using Flex, Bison and LLVM.

The compiler implements the complete compilation pipeline, from lexical analysis to LLVM IR code generation.

## Features

- Lexical analysis (Flex)
- Syntax analysis (Bison)
- Abstract Syntax Tree (AST)
- Symbol Tables
- Semantic Analysis
- Type Checking
- Method Overloading
- LLVM IR Code Generation
- Error Detection and Reporting

## Technologies

- C
- Flex
- Bison
- LLVM
